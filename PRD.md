# PRD: 拼形蛇 Shape Snake

## 1. 概述

| 项目 | 内容 |
|------|------|
| 产品名称 | 拼形蛇（Shape Snake） |
| 定位 | 移动端轻量益智游戏 |
| 场景 | 黑客松展示，用户单次体验 < 1 分钟 |
| 核心理念 | 贪吃蛇骨架 × 拼图填充玩法 |

## 2. 目标用户 & 体验目标

- **用户画像**：黑客松评委、现场观众、移动端随手玩
- **体验目标**：打开即玩 → 15-30 秒理解规则 → 20-40 秒通关 → 获得爽感
- **设计原则**：零学习成本、单次体验闭环、视觉反馈饱满

## 3. 核心规则（当前实现）

### 3.1 开局状态
- 14×14 网格
- 蛇初始长度 5，位于左侧
- 目标图形为十字形（5 格），中心在 (7,7)
- 总资源守恒：蛇长 + 吐块数 = 初始蛇长

### 3.2 操作定义
| 操作 | 效果 | 步数 |
|------|------|------|
| 转向 | 改变蛇头朝向 | - |
| 移动 | 蛇在朝向上自动前进 1 格（420ms/步） | +1 |
| 吐方块 | 从蛇尾放下一个静态方块，蛇长度 -1 | +1 |

### 3.3 通关条件（三项全满足）
1. 目标图形每格都被占用（蛇身或吐块）
2. 目标区域外无占用格
3. 占用总格数 = 目标格数

### 3.4 回收机制
- 蛇头经过已吐出的方块时自动回收，方块消失，蛇长 +1
- 资源可循环利用，避免死局

## 4. 操作方式

| 平台 | 转向 | 吐块 |
|------|------|------|
| 移动端 | 滑动屏幕（四向滑动） | 双击屏幕 / 点击「吐方块」按钮 |
| 桌面端 | 方向键 / WASD | 空格 / Enter / 点击按钮 |

## 5. 当前技术实现

### 5.1 技术栈
- **渲染**：Canvas 2D + requestAnimationFrame
- **构建**：Vite 8
- **依赖**：无第三方运行时依赖，单文件实现
- **包管理**：npm（仅 devDependency: vite）

### 5.2 动画效果（已实现）
- 蛇身 HSL 渐变色（头亮红 → 尾暗红）
- 蛇头圆角矩形 + 眼睛随方向移动瞳孔
- 蛇头运动残影拖尾（最近 8 帧轨迹）
- 目标区域脉动虚线边框
- 填充目标格时脉冲圈扩散 + 粒子喷射
- 撞墙/撞自己时屏幕震动
- 吐方块时弹出动画（缩放弹跳）+ 粒子效果
- 通关时全屏闪白 + 持续粒子喷发 1.5 秒
- 背景网格呼吸光晕

### 5.3 响应式
- Canvas 自适应屏幕宽度（min(420px, 100vw-28px)）
- 三档断点适配（>480px / ≤480px / ≤380px）
- 移动端字号、间距、按钮尺寸逐级缩放
